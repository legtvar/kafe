# v0.2.0: New Permission Model

> Originally released on 2023-12-13, but this text has been written on 2025-12-19.

## Changelog

- Added a video conversion deamon.
- Replaced the old capabilities-based permission system a recursive system implemented through SQL functions.
- Set up a regular DB backup.
- Removed the unnecessary abstraction of service interfaces.
- Added the `Err<T>` type.
